# klearn

klearn is a Python module that elaborates on scikit-learn. It provides data preprocess, tranforming, or even model ensembling tools.

* Basic Structure of klearn
    * eda
    * ensemble
    * model_selection
    * models
    * preprocessing
    * logger
    * utils
